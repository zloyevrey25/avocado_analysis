# 🥑 Avocado Price Analysis

## 📌 Описание проекта
В данном проекте проведён разведочный анализ (EDA) датасета цен на авокадо Hass.

Цель — изучить распределение цен, выявить зависимости и подготовить данные для построения моделей машинного обучения.

---

## 📊 Данные
Используется датасет:
- Источник: Kaggle
- Объекты: продажи авокадо
- Основные признаки:
  - AveragePrice — средняя цена
  - Total Volume — объём продаж
  - type — тип (organic / conventional)
  - region — регион

---

## 🔍 Этапы анализа

### 1. Предобработка данных
- Проверка пропусков
- Преобразование даты
- Кодирование категориальных признаков

### 2. Анализ распределений
- Гистограммы цен
- Анализ выбросов
- Проверка нормальности

### 3. Взаимосвязи признаков
- Pairplot
- Корреляционная матрица

### 4. Подготовка к ML
- Нормализация данных
- Разделение на train/test
